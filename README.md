# 4-Player Chess Clock & Score Tracker

A mobile-optimized web application for 4-player chess (or any 4-player board game) that handles timing and scoring.

## Features

- **4-Player Clock**: Independent timers for Red, Blue, Yellow, and Green players.
- **Clockwise Turn Order**: Automatically switches to the next player.
- **Scoring System**: Track points for rules like checkmate (+20), captures, and more.
- **Customizable**: Change player colors, time controls (3, 5, 10, 15+ min), and increments.
- **Mobile First**: Designed as a PWA-like experience for phones/tablets on the table.
